### Summary

When uv runs a script via its file path, sys.path doesn't include the repo root by default. Fix: inject `Path(__file__).parent.parent` into `sys.path` before the `systems.X` import. Why `cost_dashboard.py` didn't hit Bug 2: its `systems.X` import is inside `fetch_coverage_report` (lazy, only fires on `--coverage` flag). Only `run_optimizer_weekly.py` + `provision_new_client.py` had the top-level import. Fixed both preemptively.
