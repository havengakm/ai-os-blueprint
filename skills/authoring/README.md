# Authoring skills

Skills that create or update content — templates, variants, offers, sequences, knowledge entries.

## Planned skills

- `compose-draft.md` — given a contact + niche + offer + round, bandit-select one variant per component type, fill placeholders, write `outreach_drafts` row with `component_selections` JSONB. (Plan 1 Task 15)
- `write-component-variant.md` — operator-facing: add a new subject line / icebreaker / pain_hook / offer_frame / CTA / signature variant to the component registry. Walks operator through offer-score-27-constraints validation. (Plan 1 Task 13)
- `score-offer-27-constraints.md` — run a proposed offer (or variant) through the 27-constraint offer-score rubric, return pass/fail per constraint with rationale. Reference `feedback_offer_score_framework`. (Plan 2)
- `write-yaml-sequence.md` — author a new sequence YAML for a niche-round-angle combination. Validates DAG integrity, channel availability, event triggers. (Plan 2)
- `refine-knowledge-entry.md` — operator-facing: add or update an expert-framework entry in the knowledge base. Re-embeds on save. (Plan 7 — most authoring is pre-launch)
- `propose-new-niche-config.md` — when a client's outreach performance suggests a new niche is worth carving out, propose the ICP + weights + tier_thresholds delta. Operator approves. (Plan 7)
