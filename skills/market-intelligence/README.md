# Market Intelligence skills

Atomic skills for understanding the customer, the market, and the voice of the customer. Every skill: one input → one output.

## Planned atomic skills

### Customer Research
- `extract-icp-traits.md`: input: customer list → output: common firmographic + behavioural traits
- `segment-personas.md`: input: profiles → output: 3-5 distinct personas
- `extract-pain-points-from-transcripts.md`: input: call transcripts → output: ranked pain points with frequency
- `map-jobs-to-be-done.md`: input: customer context → output: functional + emotional jobs
- `identify-buying-triggers.md`: input: events + data → output: trigger → likely action mapping
- `cluster-objections-from-data.md`: input: sales conversations → output: objection clusters
- `estimate-willingness-to-pay.md`: input: customer + market data → output: price range with reasoning
