# General (Unattributed Frameworks)

Frameworks and patterns that aren't tied to a single named expert. Use when the pattern is widely shared across the field.

## Current files

- `copywriting-frameworks.md`: AIDA, PAS, FAB, and other foundational structures (migrated from flat `data/knowledge/` on 2026-04-22)
- `business-frameworks.md`: general business frameworks (migrated from `context/` root on 2026-04-22)

## When to put something here vs a named expert folder

If one named expert's name is shorthand for this framework (Hormozi's Grand Slam Offer, Brunson's Perfect Webinar), file it under that expert. If the framework has many parents or none (AIDA, PAS, MEDDIC, BANT, Porter's Five Forces), file it here.
